## Changes Summary (Username Duplication Check)

### 1. `AuthService.java`

- `유저네임중복체크(String username)` 메서드를 추가했습니다.
- `UserRepository`를 통해 유저네임을 조회하고, 이미 존재하는 경우 `Exception400`을 발생시켜 클라이언트에 알립니다.

```java
public void 유저네임중복체크(String username) {
    userRepository.findByUsername(username).ifPresent(user -> {
        throw new Exception400("동일한 유저네임이 존재합니다");
    });
}
```

### 2. `AuthController.java`

- `GET /check-username` 엔드포인트를 추가했습니다.
- `username`을 요청 파라미터로 받아 `AuthService`를 호출합니다.
- 예외가 발생하지 않으면, 사용 가능한 유저네임이라는 성공 응답을 반환합니다.

```java
@GetMapping("/check-username")
public ResponseEntity<?> 유저네임중복체크(@RequestParam String username) {
    authService.유저네임중복체크(username);
    return Resp.ok("사용가능한 유저네임입니다");
}
```
